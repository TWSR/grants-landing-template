這個專案是什麼？

本專案透過三軸感測器記錄用路人車輛的晃動程度為體感數據，並經由社群參與及統計分析，依此推估行駛道路的（不）平坦程度，以供政府單位作為參考，並期待道路改善提升用路人觀感。

簡單採集回報

透過手機App背景下運作，蒐集道路不平坦資訊。

雲端分析處理

將資料回傳伺服器後，以雲端方式處理。

成果展示與共享

以ＧＩＳ圖台方式展示成果並透過開放平台共享。

為什麼要做這個專案？

看到每年政府花費的在道路修補上的費用與自身道路駕駛感受不太相符，希望能找一個方式，讓主管單位能証明修路預算花在該花的地方，另一方面則是讓用路人抱怨無感時有所佐証。

目前成果 (不定期更新)

3月工作

  • Project system architecture design
  • Hardware equipments procurement including Respberry Pi & Andriod Phone for web-app development.
  • Cloud environment setup for data transforming
  • Project schedule refined based on g0v requirement

4月預期工作(及進度回報)

  • Web-based app development and alfa test (目前進度：Alfa版完成，可收集原始三軸加速器的紀錄資料並回傳伺服器。)
  • Algorithm development - Road Flatness Relative Ranking Design （目前進度：將回傳之資料進行統計及分類，並透過標準差過大的數值視為道路不平坦之特徵。）
  • Field data collection for verification of algorithm design （目前進度：已陸續收集近百公里數據，含高速公路，省道及一般市區道路並使用於初版道路不平坦指標開發;內部資料分析展示平台初版開發完成）
  • Identify and interview with the primary users such as government agency, college and private company （目前進度：已連繫大學教授及公路機關進行初步意象溝通）

5月預期工作

  • Fixmystreet 平台環境設計導入
  • 不同手機平台之三軸加速器的現地資料採集
  • 道路不平坦演算法優化及驗證
  • 外部顧問：與大學教授訪談, 與公路機關洽談
